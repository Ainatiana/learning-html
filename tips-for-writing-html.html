Tips for writing HTML pages

Do you want to be a fast HTML writer?

  1. Firstly, think about what should be your work when it’ s finished. Then draw a picture of it in your mind (its appearance).
  2. When typing, use copy ctrl+c and paste ctrl+v when useful.
  3. Tape the closing tag right after taping the opening tag then put the content inside those tags afterward to make your job easier.
  4. Don’ t usually rely on using the mouse, use the click board instead.
  5. Use click board short-cuts. Such as shift+fn +endorhome.

Tools we need to write an HTML document:

  1. Make a template in another file called template.html and use the right HTML elements.
  2. Do not forget (.html) at the end of your filenames. For example: filename.html.
  3. Test passing your document through one or more browsers in order to make sure that it’ s working well.
  4. Have an index file in your html document and name it index.html as your homepage.
  5. Also, have a readme.html file in your document.

Make sure your work is high quality by:

  1. Make your code easy to read by using good indention so it’ s not complicated for other to read.
  2. Using the tab to make the right indention.
  3. Pass your work through the data validation to made sure that there is no error in your document.
