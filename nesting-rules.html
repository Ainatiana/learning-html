The nesting rule

How do we nest element inside another element?

Nesting means put one or more elements inside other elements. We define those elements as child or children element and parent elements. Some elements can' t be children of some elements so pay attention.These examples below will let you know more about the nesting rule:

You cannot do this:

<p><h2>This one is rubbish</h2></p> or <h2><p>This one is rubbish too</p></h2>

Those are example of error because you cannot have the <h> element as a child of the <p> element nor <p> element as a child of the <h> element.

Do this instead:

<section><p>This is perfect</p></section>

You need to make them written individually as in the example above.
