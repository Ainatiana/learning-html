Rules for HTML5:

These rules below will help you writing successful HTML5 document:

  • Every HTML element is defined by an opening tag and a closing tag wrapped around their content, pair of angle brackets surround each tag.
  • You have to start typing your code by <DOCTYPE html>
  • The difference between the opening tag and a closing tag is a single forward slash (/) for a closing tag.
  • It' s not always necessary to put closing tag to all elements because some HTML5 elements don't need to have closing tag.
  • You can make an empty element close itself by adding a forward slash (/) before the second angle bracket (>)
  • Technically, you could mix uppercase and lowercase letters in HTML5. However, it's generally deemed good practice to use lowercase letters for element and attribute names.
  • You can add a blank line to separate large code blocks, but don't overuse them.
  • White space is tricky too. You can add two spaces for indentation
  • Code comments are written between an opening angle bracket followed by an exclamation mark and two dashes and two dashes then aclosing angle bracket. They do not affect the web page on the user side, as their purpose is to only make notes for the developer
