Attribute rules:

Here are the rules about attribute that you have to follow:

Attributes act like extra information tied to an HTML element. They are written within an HTML tag. For example, the href attribute used to define the target of a link (which uses an anchor tag): <a href="https://this-website.com">this-link</a>

Some HTML elements have obligatory attributes. For example, when inserting an image, you have to provide the location of the image, using the src attribute. <img src="#" alt="Description of the image"> in the src="#", the "#" is the value of the attribute "src"

  • Attributes have to be written in lowercase.
  • Attribute value must have quotes.
  • Attributes are separated with spaces.
  • You cannot have quotes (“”) inside quotes, use (‘) instead.
  • You have to have value inside the quotes.
  • Id attribute identifies an element in HTML document.
  • Once a name is used as the value of an id attribute of an HTML element, it can not be used as the value of any other element's id attribute in that document.
  • Unlike id, name of a class can be used with multiple elements in an HTML document.
  • Attribute names must consist of one or more characters other than the space characters.
  • Attribute value must not contain space.
  • It is necessary to add alt atribute to image because sometimes, the image for some reason cannot be displayed.
  • If it' s possible, try not to put any space around equal signs.

Hope those rules are helpful for you to write better attributes in your document.
