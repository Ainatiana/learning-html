Definition:

HTML element:

When referring to Hypertext Markup Language (HTML), the code is made up of sets of characters between angled brackets; often referred to as tags. The complete tag including the opening tag, enclosed tag content, and the closing tag is the HTML element. An element may also tell the browser how to display text. For example, text wrapped in the <b> bold element, denoted with displays a bit darker, whereas the <i> italic element makes text appear italicized.
